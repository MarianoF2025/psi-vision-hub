# Centralwap - Overview

## ¿Qué es Centralwap?

Centralwap es una **plataforma de gestión de comunicaciones WhatsApp** que funciona como una central telefónica digital para empresas que venden y atienden clientes por WhatsApp.

### Analogía: La Central Telefónica

Imaginá una empresa con un único número de teléfono principal:

1. Cliente llama al número
2. Escucha: "Presione 1 para Ventas, 2 para Soporte..."
3. Se conecta con el área elegida
4. El agente atiende desde su interno
5. **La llamada sigue siendo por el número principal**

Centralwap hace exactamente esto, pero para WhatsApp.

## ¿Qué problema resuelve?

### Antes de Centralwap

| Problema | Consecuencia |
|----------|--------------|
| Cada vendedor usa su WhatsApp personal | No hay control ni historial centralizado |
| Vendedor se va = pierde contactos | Pérdida de leads y relaciones |
| No saben cuántos leads llegan | Imposible medir ROI de publicidad |
| Respuestas diferentes a cada cliente | Experiencia inconsistente |
| Meta Ads genera leads sin seguimiento | Dinero tirado |
| Múltiples herramientas desconectadas | Datos fragmentados |

### Con Centralwap

| Solución | Beneficio |
|----------|-----------|
| Bandeja unificada para todo el equipo | Control total de comunicaciones |
| Historial permanente en la nube | Continuidad independiente del personal |
| Métricas en tiempo real | Decisiones basadas en datos |
| Respuestas rápidas estandarizadas | Experiencia consistente |
| Integración automática CTWA | Cada lead de Meta Ads se captura |
| Todo en un solo lugar | Single source of truth |

---

## Principio Fundamental

> **"Por donde entra, sale"**

Un mensaje que entra por una línea WhatsApp siempre sale por esa misma línea, independientemente del área o agente que lo atienda.

Esto garantiza:
- El cliente siempre ve el mismo número
- No hay confusión con múltiples contactos
- Historial unificado por conversación

| Módulo | Función |
|--------|---------|
| **CRM** | Interfaz para agentes - chat, contactos, respuestas rápidas |
| **Router** | Menús IVR, derivación automática entre áreas |
| **Automations** | Menús Click-to-WhatsApp, autorespuestas |
| **Remarketing** | Campañas masivas, segmentación |
| **Grupos** | Gestión de grupos WhatsApp |
| **Analytics** | Métricas, reportes, exportación |

## Modelo de Servicio

Centralwap se ofrece como **servicio gestionado**:

| Concepto | Incluye |
|----------|---------|
| **Setup Inicial** | Instalación, configuración, migración, capacitación |
| **Fee Mensual** | Hosting, mantenimiento, soporte, actualizaciones |

### Ventajas del modelo

- Sin inversión inicial en infraestructura
- Soporte técnico incluido
- Actualizaciones automáticas
- Escalable según necesidad
